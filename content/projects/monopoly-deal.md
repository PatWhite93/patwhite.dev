A full implementation of the Monopoly Deal card game as a real-time mobile application. Features include:

- Real-time 2+ player gameplay
- Multiplayer lobby with game discovery
- Concurrent game state management
- Complete rule enforcement for all card types

Built with React Native and Firebase Realtime Database for instant state synchronization across players.
